## 📋 衝突仲裁 SOP

當發生跨專案資源衝突時：

1. **收集資訊**：各方提供需求背景與急迫性
2. **評估框架**：使用 `.ai/principles.md` 框架 A
3. **仲裁決策**：PM 總管做出決策
4. **記錄決策**：於 `5_management/decision_records/` 建立文件
5. **更新此文件**：記錄衝突與解決方式
6. **同步各方**：通知相關子 PM 與利害關係人
