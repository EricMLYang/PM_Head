## 🎯 使用方式

1. 複製對應範本到目標資料夾
2. 重新命名為 `YYYY-MM-DD_[標題].md`
3. 填入實際內容
4. 依照範本的結構保持一致性

## 📊 報告寫作原則

### 週報
- **簡潔明確**：每個產品線 2-3 句話
- **突出異常**：重點放在需要關注的議題
- **可執行**：下週計畫要明確可追蹤

### 月報
- **數據驅動**：用數據說話，展示趨勢
- **策略視角**：不只是狀態搬運，要有分析
- **前瞻性**：提供下月的策略建議

### 季報
- **高階摘要**：決策層級的人要 5 分鐘看完
- **商業影響**：強調對業務的實際價值
- **策略調整**：基於本季執行結果的策略建議

### 決策記錄
- **脈絡完整**：半年後回頭看還能理解為什麼這樣決定
- **選項透明**：記錄所有考慮過的選項
- **可追蹤**：明確行動項目與負責人
